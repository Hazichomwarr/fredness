# Frednes International Market 🛒

A modern e-commerce website for an African & Caribbean wholesale and retail food store based in Orange, NJ.

## 🌍 Overview

Frednes International Market provides authentic African and Caribbean food products for both individual customers and bulk buyers. This platform is designed to deliver a smooth shopping experience while highlighting wholesale capabilities.

## ✨ Features

- 🟢 Clean, modern landing page
- 🛒 Product browsing by category
- 🔥 Best sellers section
- 🍲 Meal bundles (Jollof, Egusi, etc.)
- 🏪 Wholesale & retail positioning
- 📍 Google Maps integration
- ⚡ Fast and responsive UI

## 🧱 Tech Stack

- Next.js
- React
- Tailwind CSS
